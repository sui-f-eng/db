安装步骤

  1. 下载扩展文件
    点击上方按钮下载 zip 压缩包，解压到本地文件夹。
  2. 打开 Chrome 扩展管理
    在地址栏输入 chrome://extensions 并回车。
  3. 开启开发者模式
    页面右上角打开「开发者模式」开关。
  4. 加载扩展
    点击左上角「加载已解压的扩展」，选择刚才解压的文件夹。
  5. 获取 API Key
    首次使用按快捷键触发截图，会弹出输入框，填入豆包 API Key 即可。
